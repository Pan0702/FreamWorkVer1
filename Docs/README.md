# Framework 説明書（クラスリファレンス）

`Src/` と `Shaders/` の各ソースファイルを、フォルダツリーから辿って
**ファイルごと・クラスごと**に解説を読める HTML ドキュメントです。

## 開き方

`Docs/index.html` を**ブラウザでダブルクリックして開くだけ**。
サーバーもネット接続も不要（1 ファイルで完結しています）。

- 左のツリーで `Src` → `Engine` … と展開し、ファイルをクリックすると
  右側にそのファイルの解説（定義されている型・関数・メンバ変数）が出ます。
- フォルダ名をクリックすると、そのレイヤーの概要と含まれるファイル一覧が出ます。
- 右上の検索でファイル名・クラス名を絞り込めます。
- 右上の ◐ ボタンでライト/ダークを切り替えられます。

## 中身はどこから来るのか

解説文は**手書きではなく、ソース中の Doxygen コメント**
（`/** @brief ... */` や `//` コメント）と、クラス定義・関数シグネチャ・
メンバ変数（`_` 終わりの規約）から**自動抽出**しています。
つまり「コードのコメントを充実させる」＝「説明書が充実する」という関係です。

## 新しいファイルを追加したとき（追記のしかた）

1. いつもどおりソース（`.h` / `.cpp` / `.hlsl`）を追加し、
   クラスや関数に Doxygen コメントを書く。
2. 次のコマンドを 1 回実行するだけ：

   ```
   python Docs/generate.py
   ```

   → ツリーにも解説にも自動で反映され、`Docs/index.html` が再生成されます。

手作業で HTML を編集する必要はありません。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `Docs/index.html` | 生成物。これを開く（Git 管理対象）。 |
| `Docs/generate.py` | ソースを走査して `index.html` を作るジェネレータ。 |
| `Docs/_template.html` | ビューアの見た目・操作（HTML/CSS/JS）のテンプレート。 |

## 抽出範囲・仕様メモ

- 対象: `Src/`, `Shaders/`。`ImGui/` や `x64/Debug` などの外部・生成物は除外。
- レイヤー（Core / Platform / Graphics / … ）ごとに色分け。
  各レイヤーの概要文だけは `generate.py` の `LAYER_OVERVIEW` に手書きで
  書いてあるので、必要なら編集してください。
- ソースは Shift-JIS(CP932) と UTF-8 が混在していても自動判別して読みます。
